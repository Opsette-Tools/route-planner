# Route Planner

Plan and optimize multi-stop delivery routes. Built with React, Ant Design, Leaflet, and OSRM.

## Features

- Route optimization for up to 15 stops via OSRM
- Interactive map with drag-and-drop stop reordering
- Save and reuse routes (stored locally)
- Export routes as CSV
- Navigate to any stop via Google Maps
- Installable as a PWA

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to GitHub Pages via the included workflow. Push to `main` to trigger a deploy.
